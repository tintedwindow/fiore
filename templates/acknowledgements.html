Thank you for making this possible
This is a CS50 project.
Perhaps, the course that shaped me as a programmer, and more so a problem solver ...
When I started thinking of a project idea, initially I wanted to create a mini version of Geoguessr because I loved it so much (basically a game where you are placed in a random street view around the world and you have to guess where you are), and when I saw it was possible via OpenGuessr, I was eager to start my final project for CS50. However, I soon realized that Geoguessr relies on Google Street View, and using it's API requires a Google Cloud account, which involves billing and a credit card- something I didn't have at that time. This was a frustrating setback, especially since the idea to create a mini Geoguessr had excited me so much.
It was also around the time Apple was having one of their WWDC events, and me having watched that, YouTube was recommending me older WWDC videos. And that's when I saw WWDC 2013.
The first thing we ask is what do we want people to feel?
That line hit me. As someone who always saw exploring street view as a means of escape, a calming feeling waiting for me in a random street of the world, I knew what I wanted my project to make people feel.
I also had started journaling, thanks to Jade, Nathaniel, Sophie, Ali and Sadia. It was a mixed experience at first because I had just started, but quickly things started connecting. What made it easier to journal was on Ali's suggestion to try an app called Day One, a digital journal where on a free account, you were limited to one entry a day. And this free plan limitation surprisingly ended up helping me in simplifying my journaling habits!
This limitation and the new found simplicity ultimately landed me on a vague idea I sketched on my Keeps one night. And I thought, if I could combine the feeling of looking at a place or a memory and associating it with a day, it could work. Then came the idea for Fiore! Framing instances on rectangles everyday! This project made me realize that there's so much to learn for me and it also made me realize that I infact did when I finally solved each challange whether it was flying the nett and setting up local development and making Flask work, or going from a table tag to divisions that represent days in Fiore now :)
I have always been a Notion user, and one thing that was really inspiring to me was how they kept everything so simple yet organized. I specially loved their website, and used to think that one day I would make one like this. And that's how the design of the website got finalized. It really reciprocated with the calmness of Fiore as well. Coming to the libraries, and acknowledgements realted to code, Google Fonts provided the font Inter, and it's font variations are used across the website. I made this project on Visual Studio Code. The backend language was Python and the web framework used was Flask.
Here's all the Python libraries that hepled me make it possible.
  • cs50 (also provided SQL features)
  • flask
  • flask_sessions
  • werkzeug.security
  • werkzeug.utils
  • Built-in Python libraries: calendar, datetime, functools, logging, os, re, uuid
I ran Ubuntu in WSL. Additionally, I want to also thank Stack Overflow, the visits to which got more and more frequent as the project expanded. This project also introduced me to Git and Github, and Fiore's repository can be found here.
Once again, in this project I was standing on the shoulder of giants.
So, from the bottom of my heart, thank you!
